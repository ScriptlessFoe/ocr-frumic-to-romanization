# Frumic to Romanization

Automatically reads Frumic from a screenshot and romanizes it into letters according to the Frumic font style.

Code is currently very rough, refactor and potential web integration soon.

## How to use

- set up .venv with requirements.txt
- create "input" and "output" directories in top directory
- place reasonably sized .jpg images in "input" directory
  - size of each symbol should be around 20 by 20 pixels
- run "src/main.py" from top directory
